# Buscador de frases de Friends

En este sitio web puedes filtrar y encontrar tus frases favoritas de los personajes principales de la serie Friends. Si no encuentras la frase que buscas, puedes añadirla.

## Herramientas 🛠️
- HTML
- SCSS
- JS
- React
- NPM
- GitHub Pages
